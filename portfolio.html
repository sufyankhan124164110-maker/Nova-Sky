Project: Nova Sky

Nova Sky is a browser-based astronomy tool built from scratch using plain HTML, CSS, and JavaScript — no frameworks, no backend server. Every feature runs entirely on the device, pulling in live data only where needed.

Features

🛰 Live ISS Tracking — real-time latitude, longitude, altitude, and speed of the International Space Station.

🛰 Satellite Pass Predictions — calculates upcoming visible ISS passes for the user's exact location, including rise/set direction and duration.

🪐 Planet Tracker — computes the real-time sky position (altitude & direction) of every visible planet plus the Moon, using orbital mechanics.

⭐ Live Star Map — a simulated dome-view sky map plotting ~50 of the brightest named stars and all visible planets at their true current position, calculated from sidereal time and the observer's coordinates.

🔭 AR Sky View — an experimental augmented-reality mode that overlays live star and planet positions onto the phone's camera feed, using device compass and tilt sensors.

📍 Live Location & Clock — real-time clock and GPS-based location, used to personalize every calculation above.

Technical approach

The site uses the open-source Astronomy Engine library for precise planetary and lunar position calculations, a hand-built star catalog with sidereal-time-based coordinate transforms for the star map, and the free ISS Pass and Where The ISS At APIs for satellite tracking. Everything renders client-side on an HTML canvas — no data is sent to or stored on any server.

Status

Nova Sky is actively being developed, with new features added regularly.
